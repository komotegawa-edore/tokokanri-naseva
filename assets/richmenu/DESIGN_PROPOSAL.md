# リッチメニューデザイン提案

## 基本仕様

- **サイズ**: 2500 x 843 px
- **形式**: PNG
- **レイアウト**: 3分割（横並び）
- **ファイルサイズ**: 1MB以下

## Canvaでの作成手順（推奨）

### ステップ1: 新規デザイン作成

1. [Canva](https://www.canva.com/)にアクセス
2. 「カスタムサイズ」をクリック
3. 幅: 2500px、高さ: 843px を入力
4. 「新しいデザインを作成」

### ステップ2: 背景色を設定

1. 左メニュー「図形」→「四角形」を3つ追加
2. 各四角形のサイズを調整:
   - 左: 833 x 843px、色: #E8F5E9
   - 中: 833 x 843px、色: #FFF3E0
   - 右: 834 x 843px、色: #E3F2FD

### ステップ3: アイコンを追加

1. 左メニュー「素材」→「アイコン」で検索
   - "school" または "building"（登校）
   - "hand wave" または "goodbye"（下校）
   - "chart" または "graph"（履歴）
2. 各エリアの中央上部に配置（サイズ: 120px）
3. 色を調整

### ステップ4: テキストを追加

1. 左メニュー「テキスト」→「見出しを追加」
2. 各エリアにテキストを配置:
   - **登校** / Check In
   - **下校** / Check Out
   - **登校履歴** / History

### ステップ5: 装飾を追加（オプション）

1. 区切り線を追加（図形→線）
2. 影や効果を追加して立体感を出す

### ステップ6: ダウンロード

1. 右上「共有」→「ダウンロード」
2. ファイルの種類: **PNG**
3. 「ダウンロード」

## サンプルテンプレート

もし時間がない場合、以下のコマンドで簡易的なテンプレートを作成できます：

```bash
# ImageMagickをインストール（Homebrewの場合）
brew install imagemagick

# シンプルなリッチメニュー画像を生成
convert -size 2500x843 xc:white \
  -fill '#E8F5E9' -draw 'rectangle 0,0 833,843' \
  -fill '#FFF3E0' -draw 'rectangle 833,0 1666,843' \
  -fill '#E3F2FD' -draw 'rectangle 1666,0 2500,843' \
  -fill black -pointsize 60 -font Arial-Bold \
  -draw "text 350,450 '登校'" \
  -draw "text 1183,450 '下校'" \
  -draw "text 2000,450 '履歴'" \
  richmenu.png
```

## よくある質問

### Q: 画像のサイズが分からない

**A:** 画像ファイルのプロパティで確認できます。
- Mac: ファイルを選択 → `Cmd+I`
- Windows: ファイルを右クリック → プロパティ

### Q: 1MBを超えてしまう

**A:** 画像を圧縮してください。
- オンラインツール: [TinyPNG](https://tinypng.com/)
- 品質を少し下げる（Canvaのダウンロード設定で「標準」を選択）

### Q: デザインが苦手です

**A:** シンプルな案1で十分です。
- 背景色を薄く
- アイコンは絵文字でOK（📍👋📊）
- テキストは太字で大きく
